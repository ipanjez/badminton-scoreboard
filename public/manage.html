Cara Menjalankan Multi Court

# Dari folder Badminton:
node -r dotenv/config server.js dotenv_config_path=.env.court1
node -r dotenv/config server.js dotenv_config_path=.env.court2
node -r dotenv/config server.js dotenv_config_path=.env.court3

Atau klik tools/MULAI.bat → pilih opsi 2 untuk menjalankan semua sekaligus.
