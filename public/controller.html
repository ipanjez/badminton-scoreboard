❓ Panduan Wasit / Operator

⚙ SETUP PERTANDINGAN

Klik tombol ⚙ Setup di atas kanan. Isi nama turnamen, kategori, lapangan, nama pemain, klub, wasit. Pilih siapa yang servis pertama. Klik Simpan & Terapkan.

MENAMBAH POIN

Tekan tombol + besar sesuai tim yang memenangkan reli. Perpindahan servis (Service Over) terjadi otomatis.

↩ KOREKSI / UNDO

Tekan tombol ⟵ Koreksi atau ↩ Undo untuk membatalkan poin terakhir. Bisa dilakukan berkali-kali.

🏸 SERVIS MANUAL

Tekan tombol Jadikan Serve untuk mengatur servis secara manual jika terjadi kesalahan. Biasanya tidak perlu — servis berpindah otomatis.

⏱ INTERVAL (Skor 11)

Banner kuning INTERVAL muncul otomatis saat skor 11 pertama kali tercapai. Istirahat 60 detik, lalu klik Lanjutkan ▶.

▶ PINDAH GAME

Setelah satu game selesai, klik Game ▶ Lanjut. Ganti servis awal game baru secara manual dengan tombol Serve jika perlu.

📄 EXPORT PDF

Klik 📄 PDF kapan saja untuk mengunduh score sheet resmi format BWF. PDF berisi riwayat skor lengkap, lingkaran merah di poin 11 & poin kemenangan, dan kolom tanda tangan wasit.

🔄 RESET MATCH

Klik 🔄 Reset untuk memulai pertandingan baru. Semua skor & riwayat akan dihapus. Muncul konfirmasi sebelum dieksekusi.

💡 Tips: Jika tampilan TV tidak update, minta penonton/operator TV untuk me-refresh browser (F5 atau Ctrl+R). Pastikan semua perangkat terhubung ke Wi-Fi yang sama.
