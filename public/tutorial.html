PANDUAN PENGGUNAAN

Sistem Pencatatan Skor Bulutangkis

Web-Based Badminton Scoreboard & Match Manager

Persyaratan: Tablet/HP sebagai Controller & TV/Monitor sebagai Display.
Semua perangkat harus terhubung ke jaringan yang sama (Wi-Fi Router, Hotspot, atau Hybrid LAN+Wi-Fi).

📡 Bagian 1: Persiapan Jaringan (Wi-Fi / LAN)

1
Buat Hotspot di Laptop/PC Server
Di Windows: Buka Pengaturan → Jaringan → Hotspot Seluler. Aktifkan hotspot. Catat nama dan password Wi-Fi yang dibuat.
2
Hubungkan Semua Perangkat ke Satu Wi-Fi yang sama
Tablet wasit, TV (Smart TV/Android TV), dan HP penonton semuanya harus terhubung ke jaringan yang sama. Koneksi internet tidak diperlukan.
🌐 Opsi Jaringan (Pilih Satu):
Opsi A: Hotspot PC (Nirkabel)
  • PC/Laptop → Aktifkan Hotspot Seluler (Windows Settings)
  • Semua perangkat terhubung via Wi-Fi ke hotspot
  • ✓ Keuntungan: Cepat setup, tidak perlu router
  • ⚠ Catatan: Performa bisa fluktuatif jika ada gangguan Wi-Fi
Opsi B: Router Wi-Fi (Nirkabel) — ⭐ Direkomendasikan
  • Gunakan router Wi-Fi yang sudah ada (atau beli router murah)
  • PC/Server dan semua perangkat terhubung ke router yang sama
  • ✓ Keuntungan: Performa stabil, jangkauan lebih baik, dukungan lebih banyak devices
  • ✓ Rekomendasi: Opsi terbaik untuk turnamen atau penggunaan jangka panjang
Opsi C: LAN Kabel Ethernet (Hybrid)
  • PC/Server terhubung via kabel LAN ke router/switch
  • Tablet, TV, HP terhubung via Wi-Fi ke router yang sama
  • ✓ Keuntungan: Server sangat stabil (kabel), devices fleksibel (Wi-Fi)
  • ✓ Terbaik untuk: Arena besar atau lokasi dengan gangguan Wi-Fi
Perbandingan Kecepatan: LAN Kabel (Paling cepat & stabil) > Router Wi-Fi (Cepat & stabil) > Hotspot PC (Bergantung sinyal)
3
Jalankan Server di PC
Buka terminal/CMD di folder aplikasi, lalu ketik: node server.js Server akan menampilkan alamat IP jaringan, contoh: http://192.168.137.1:3000
4
Buka Alamat di Setiap Perangkat
📺 TV/Proyektor: Buka browser → http://[IP]:3000/display
📱 Tablet Wasit: Buka browser → http://[IP]:3000/controller
👀 HP Penonton: Buka browser → http://[IP]:3000/viewer

🔐 Bagian 2: Login Panel Kontrol

Halaman /controller dilindungi PIN. PIN default adalah 1234.

⚠ Keamanan: Untuk keamanan optimal, ubah PIN sebelum digunakan di turnamen resmi. Jalankan server dengan perintah:
CONTROLLER_PIN=7890 node server.js

Setelah login, sesi akan bertahan selama 12 jam. Klik tombol "Keluar" untuk mengakhiri sesi.

⚙ Bagian 3: Setup Pertandingan

1
Klik tombol ⚙ Setup di kanan atas controller.
2
Isi nama turnamen, kategori, nomor lapangan, wasit, dan nama pemain.
Nama pemain yang pernah digunakan akan muncul sebagai saran otomatis (auto-complete).
3
Pilih tim mana yang Servis Pertama, lalu klik ✓ Simpan & Terapkan.

🏸 Bagian 4: Mencatat Poin

+
Tombol + (Plus)
Tekan untuk menambah 1 poin ke tim tersebut. Perpindahan servis (Service Over) terjadi otomatis jika tim yang bukan server yang memenangkan reli.
↩
Koreksi / Undo
Membatalkan poin terakhir yang dicetak. Bisa ditekan berkali-kali untuk mundur lebih jauh.
🏸 Aturan BWF yang Diimplementasikan:
  • Rally Point 21 — menang harus unggul 2 poin
  • Deuce maksimum 30 poin (30-29 = menang)
  • Interval otomatis saat skor 11 pertama kali tercapai
  • Service Over otomatis — server berubah otomatis ke tim pemenang reli
  • Best of 3 games — tim pertama yang memenangkan 2 game
⚡ Interval:
Saat skor 11 tercapai, banner "INTERVAL" muncul di TV dan controller. Klik Lanjutkan ▶ di controller setelah istirahat 60 detik selesai.

📄 Bagian 5: Game Berikutnya & Export PDF

1
Setelah suatu game selesai, tekan Game ▶ Lanjut untuk pindah ke game berikutnya.
2
Setelah pertandingan selesai, tekan 📄 PDF Export. Browser akan langsung mengunduh file PDF resmi berformat Landscape A4 dengan layout score sheet BWF standar.
📋 Isi PDF Score Sheet:
  • Header: Nama Turnamen, Event, Match No, Court, Date/Time, Wasit
  • Tabel ringkasan skor (Game 1, 2, 3 + pemenang)
  • Grid skor per game dengan notasi kronologis BWF
  • Lingkaran merah pada poin Interval (11) dan poin kemenangan
  • Indikator Service Over (tanda diagonal)
  • Footer: Waktu mulai/selesai, durasi, kolom tanda tangan

🔧 Troubleshooting

Perangkat tidak bisa membuka alamat server
Pastikan semua perangkat terhubung ke hotspot yang sama. Coba matikan firewall Windows sementara.
Skor tidak update di TV
Muat ulang halaman /display. Pastikan koneksi Wi-Fi stabil.
PDF tidak bisa dibuat
Pastikan library jsPDF berhasil diinstall (npm install sudah dijalankan). Coba refresh halaman controller.
Auto-complete nama tidak muncul
Nama akan tersimpan otomatis setelah setup pertandingan pertama. Setelah itu auto-complete akan berfungsi.
